# 🎮 Pi Network 五子棋

基于 Pi Network 区块链的 Web3 五子棋游戏，强化 AI 算法 + 完整支付系统。

## ✨ 游戏特色

- 🤖 **强化 AI 算法**: 深度 6 层 Minimax 算法，挑战性强
- 💰 **Pi Network 集成**: 官方 SDK 支付和登录
- 📊 **实时排行榜**: 全球玩家积分排名
- 📱 **移动端优化**: 完美适配 Pi Browser
- 🎯 **一键部署**: GitHub Pages 零配置上线

## 📦 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/YOUR_USERNAME/pi-gomoku.git
cd pi-gomoku
```

### 2. 安装依赖

```bash
npm install
```

### 3. 本地开发

```bash
npm run dev
# 访问 http://localhost:3000
```

## 🚀 三步部署

1. **上传到 GitHub**: 创建仓库，上传所有文件
2. **启用 GitHub Pages**: 在仓库设置中启用 Pages 功能
3. **配置 Pi Network**: 在 Pi Developer Portal 注册应用

详细步骤请查看 [部署指南.md](./部署指南.md)

## 💰 Pi Network 功能

- **用户认证**: Pi 账户登录和用户信息获取
- **支付系统**: Pi 币支付道具和服务
- **月度奖励**: 排行榜前三名自动分发奖励

## 🎮 游戏特性

- **AI 算法**: 深度 6 层 Minimax + Alpha-Beta 剪枝

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **后端**: Node.js + Vercel Serverless Functions
- **区块链**: Pi Network SDK v2.0
- **部署**: Vercel + GitHub Actions
- **音效**: Web Audio API
- **UI 框架**: 原生 CSS + CSS Grid/Flexbox
