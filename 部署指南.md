## �🔧 第一步：准备工作

### 1.1 注册必要账户

#### GitHub 账户注册

1. 访问 https://github.com
2. 点击 "Sign up" 注册
3. 验证邮箱
4. 完成账户设置

#### Vercel 账户注册

1. 访问 https://vercel.com
2. 点击 "Sign Up"
3. **重要：选择 "Continue with GitHub"** (用 GitHub 账户登录)
4. 授权 Vercel 访问你的 GitHub

#### Pi Network 开发者账户

1. 下载 Pi Browser 应用
2. 在 Pi Browser 中访问 `develop.pi`
3. 注册开发者账户
4. 创建新应用

### 1.2 安装 Git (如果还没有)

#### Windows 用户：

1. 访问 https://git-scm.com/download/win
2. 下载并安装 Git
3. 安装时选择默认选项即可

#### Mac 用户：

```bash
# 使用Homebrew安装
brew install git

# 或者从官网下载
# https://git-scm.com/download/mac
```

#### 验证 Git 安装：

```bash
git --version
```

## 🗂️ 第二步：项目上传到 GitHub

### 2.1 创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 填写仓库信息：
   - Repository name: `pi-gomoku-game`
   - Description: `Pi Network五子棋游戏`
   - 选择 "Public" (公开)
   - **不要**勾选 "Add a README file"
4. 点击 "Create repository"

### 2.2 上传项目文件

#### 方法一：使用 Git 命令行 (推荐)

```bash
# 1. 打开命令行，进入项目文件夹
cd /path/to/your/project

# 2. 初始化Git仓库
git init

# 3. 添加所有文件
git add .

# 4. 提交文件
git commit -m "Initial commit: Pi Gomoku Game"

# 5. 添加远程仓库 (替换YOUR_USERNAME为你的GitHub用户名)
git remote add origin https://github.com/YOUR_USERNAME/pi-gomoku-game.git

# 6. 推送到GitHub
git push -u origin main
```

#### 方法二：使用 GitHub 网页上传

1. 在 GitHub 仓库页面点击 "uploading an existing file"
2. 拖拽所有项目文件到页面
3. 填写提交信息: "Initial commit: Pi Gomoku Game"
4. 点击 "Commit changes"

## 🌐 第三步：部署到 Vercel

### 3.1 连接 GitHub 到 Vercel

1. 登录 Vercel (https://vercel.com)
2. 点击 "New Project"
3. 找到你的 `pi-gomoku-game` 仓库
4. 点击 "Import"

### 3.2 配置部署设置

1. **Project Name**: `pi-gomoku-game`
2. **Framework Preset**: 选择 "Other"
3. **Root Directory**: 保持默认 `./`
4. **Build and Output Settings**:
   - Build Command: 留空
   - Output Directory: `public`
   - Install Command: 留空

### 3.3 环境变量配置

1. 在 Vercel 项目设置中点击 "Environment Variables"
2. 添加以下变量：
   ```
   PI_API_KEY=你的Pi API密钥
   NODE_ENV=production
   ```

### 3.4 部署

1. 点击 "Deploy"
2. 等待部署完成 (通常 1-2 分钟)
3. 获得部署 URL: `https://your-project-name.vercel.app`

## 🔑 第四步：Pi Network 集成

### 4.1 在 Pi 开发者门户配置应用

1. 在 Pi Browser 中访问 `develop.pi`
2. 点击 "Create App"
3. 填写应用信息：
   - App Name: `Pi五子棋游戏`
   - App URL: `https://your-project-name.vercel.app`
   - Description: `运行在Pi Network上的五子棋游戏`
4. 保存配置

### 4.2 获取 API 密钥

1. 在开发者门户中找到你的应用
2. 复制 "API Key"
3. 在 Vercel 中更新环境变量 `PI_API_KEY`

## 🆘 常见问题解决

### 问题 1：部署失败

**解决方案：**

1. 检查 `vercel.json` 配置
2. 查看 Vercel 部署日志
3. 确保所有文件都已上传

### 问题 2：Pi SDK 无法加载

**解决方案：**

1. 检查网络连接
2. 确保在 Pi Browser 中测试
3. 检查 Pi API 密钥配置

### 问题 3：音效无法播放

**解决方案：**

1. 确保音频文件已上传
2. 检查文件路径
3. 测试不同浏览器
